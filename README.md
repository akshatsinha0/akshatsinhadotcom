# Akshat Sinha Portfolio

Personal portfolio website for Akshat Sinha. It shows an about section, experience, projects, skills, certifications, awards, an interactive terminal, and a contact form.

Built with Angular. The codebase is an Angular workspace: the `portfolio` application plus three libraries under `projects/` (`core` for tokens, models, and constants, `data` for content, `shared-ui` for shared components).

## Run it

From the repository root:

```bash
bun install
bun run start
```

Then open the URL printed in the terminal (default `http://localhost:4200`).

## Build

```bash
bun run build
```

Output goes to `dist/portfolio`.
